# NOURI - Gestión Inteligente de Alimentos

NOURI es una landing page moderna diseñada para una aplicación de gestión de inventario de alimentos. El objetivo es ayudar a los usuarios a reducir el desperdicio, ahorrar dinero y organizar su hogar de manera eficiente mediante alertas inteligentes de vencimiento.

## 🚀 Características

- **Gestión de Inventario**: Visualización clara de productos y sus fechas de vencimiento.
- **Alertas Inteligentes**: Notificaciones para consumir alimentos antes de que caduquen.
- **Optimización de Ahorro**: Análisis del ahorro mensual mediante la reducción de desperdicios.
- **SEO Optimizado**: Estructura de metadatos completa en español para mejor posicionamiento.
- **Diseño Responsivo**: Experiencia fluida en dispositivos móviles y escritorio.

## 📂 Estructura del Proyecto

La landing page incluye varias secciones y páginas independientes:
- **Inicio**: Landing principal con beneficios, cómo funciona y precios.
- **Sobre Nosotros**: Nuestra misión y visión por la sostenibilidad.
- **Contacto**: Formulario y canales de soporte.
- **Legal**: Páginas de Términos y Condiciones y Política de Privacidad.

## 🛠️ Tecnologías Utilizadas

- **React** + **Vite**
- **Tailwind CSS** (Estilizado)
- **Framer Motion** (Animaciones)
- **Lucide React** (Iconografía)

## 🏃 Ejecución del Código

1. Instalar las dependencias:
   ```bash
   npm install
   ```

2. Iniciar el servidor de desarrollo:
   ```bash
   npm run dev
   ```

3. Construir para producción:
   ```bash
   npm run build
   ```
